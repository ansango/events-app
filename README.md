## Getting Started <a name = "getting_started"></a>

```bash
git clone
```

### Prerequisites

- node && npm
- yarn


### Installing

#### client

```
yarn || npm i
```

#### server
```
yarn || npm i
```

### Running

#### client

```
yarn start|| npm start
```

#### server
```
yarn dev|| npm run dev
```
